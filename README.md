## 📌 Project Overview
This project is a secure file sharing system built using **Python Flask**. It allows users to upload and download files securely using AES encryption. The goal is to simulate real-world secure data exchange scenarios such as in healthcare, legal, and enterprise environments.

---

## 🔐 Security Features
- AES (Advanced Encryption Standard) file encryption before storage.
- Encrypted file transfer between users.
- Secure key management.
- Protection against unauthorized file access.
- Input validation and basic upload security checks.

---

## 🛠️ Tech Stack
- Python 3
- Flask (Web framework)
- HTML/CSS (Frontend UI)
- PyCryptodome (Encryption)
- SQLite (or file-based) for metadata storage
